# Assignment 3 - Matplotlib Data Visualisation

## Overview
This assignment covers various types of data visualisation using **Matplotlib**, **Pandas**, and **NumPy** in Python.

## Requirements
```
pip install matplotlib pandas numpy
```

## Folder Structure
```
Assignment3_Matplotlib/
├── Q1_Alphabet_LineChart/       → Line chart (Alphabet Inc. stock data)
├── Q2_Horizontal_Bar/           → Horizontal bar chart (Students vs Marks)
├── Q3_MonthSales_LineChart/     → Line chart (Year-wise sales)
├── Q4_Customised_LineChart/     → Customised line chart (Month names, dashed lines)
├── Q5_Department_BarChart/      → Bar chart (Compu-Tech departments)
├── Q6_Population_Literacy/      → Scatter + Bar chart (Population vs Literacy)
├── Q7_Marks_Boxplot/            → Boxplot (Student marks in 5 subjects)
├── Q8_Grouped_Bar_Gender/       → Grouped bar chart (Scores by gender)
├── Q9_PieChart_Subplots/        → Pie chart + Subplots (Product sales)
└── Q10_Profit_Histogram/        → Histogram (Monthly profit ranges)
```

## How to Run
Open any folder and run the `.py` file:
```
python q1_alphabet_line_chart.py
```
Each script saves its chart as a `.png` file in the same folder.
